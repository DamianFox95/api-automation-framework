# API Automation Framework

Этот проект — фреймворк для автоматизации тестирования REST API, созданный в рамках обучения SDET.

## Цель
Продемонстрировать навыки:
*   Работы с Git и GitHub
*   Написания чистого, поддерживаемого кода на Python
*   Создания структуры автотестов с использованием Pytest
*   Работы с CI/CD (GitHub Actions)
*   Генерации отчетов (Allure/HTML)

## Технологии
*   Python 3.8+
*   Pytest
*   Requests
*   Allure-pytest
*   Docker (в будущем)
*   GitHub Actions
